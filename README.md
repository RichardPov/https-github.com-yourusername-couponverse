## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/couponverse.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the server:
    ```bash
    npm start
    ```
4.  Visit `http://localhost:3000`
